# CloudNest

CloudNest is a production-grade cloud storage and collaboration platform built with Java Spring Boot, PostgreSQL, Redis, MinIO, and React.

## Vision

To build a scalable cloud storage backend using engineering practices followed by modern startups.

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Redis
- MinIO

### Frontend
- React
- TypeScript
- Tailwind CSS
- TanStack Query

### DevOps
- Docker
- Docker Compose
- GitHub Actions
- Nginx
